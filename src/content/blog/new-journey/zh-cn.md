# New Journey 

对未来的真正慷概，是把一切献给现在。       ——加缪

---
# 过去

:::quote 
记忆如潮水般缓缓褪去，

只留我独自在遗憾的沙滩上，

呆滞，哭泣。
:::

有句话我很喜欢，“往事总是在回忆时被赋予意义。”我第一次与这句话相识，是高考前的语文测
试，那篇叙事性的散文写得十分真实，可见作者对往事的记忆有多深刻。初读时，年少无知的我用懵懂的思考去理解这句话，只是想象着年迈的我回忆过去的年华会是怎样的悲欢与忧乐。那时候，天真的我想着，无论人生起伏，无论道路或是平坦或是坎坷，未来的我回忆往事时，会是退去一切光环，卸下所有包袱之后的畅快与释然。过往的一切成功与失败，得到与失去，希望与绝望，快乐与痛苦都是属于自己的财富，当回望这一切时，已然可以说，人生这一遭，不枉我来过。

正如往事在回忆时被赋予意义，回忆是连接过去、现在与未来的桥梁。过去经历过回忆时才会更加圆满，未来也因为现在的回忆而变得不同。时间可能并不是线性的叙事，我们以为顺着光阴向前走，或许只是在记忆的褶皱里捡拾片段。

人总是念旧的，我常常听着父辈们回忆年轻时的往事，他们谈起那些泛黄岁月时眼中闪烁的光芒，仿佛时光从未走远。那时候我总在想，回忆真是个神奇的东西，能够跨越时间的长河，能够熬过时间的侵蚀，但是竟又如此平常，如娟娟细流。

但是，对我而言，回忆似乎成了可望而不可求的东西。

和朋友聊天时，提起三年前的往事，对方说得眉飞色舞，我在脑海里拼命回想，却只有模糊的印象，就像一团雾把世界笼罩，让人茫然，无奈。我并不喜欢这样模糊的记忆，这种可望而不可即的感觉如同面对深渊般无力。

这并不是个例。我早就发现自己是个健忘的人，我常和朋友自嘲说记忆只有七秒。或许是因为我总习惯把生活塞得太满，用各种各样的事情填满每一个缝隙，却忘了怎样去真正感知当下。日子呼啸而过，我忙着赶路，却未曾驻足看看路边的风景。于是到头来，回忆成了一件奢侈的事。

往事就这样如流水般溜走，我曾以为终将饱满的回忆，如今却成了最大的遗憾。

我开始害怕，害怕未来的我，会因记忆的模糊而陷入更深的遗憾。我必须做些什么，来拯救那正在沦陷的灵魂。

这便是这个 Blog 的由来。愿这片角落，能为我留住那些正在消逝的记忆。

# 现在

:::quote
从今天起，做一个幸福的人，
喂马，劈柴，周游世界。
从今天起，关系粮食和蔬菜，
我有一所房子，面朝大海，春暖花开。

——海子
:::

我开始学着收拾行囊。不是物理意义上的行李，而是生命中正在发生的时刻。

一年前，我曾尝试用日记留住生活，在无数个深夜写下当天的琐碎。可那些文字大多停在开头，像一封封未寄出的信，静静躺在书架的角落。我总想着“明天再补”，可明天又堆满了新的明天。日记就此搁笔，那些文字终究没能追上今天。

而这个博客，便是我为自己搭建的一间小屋。它不需要华丽，不需要被许多人看见。它只需要足够安静，能让我在喧嚣褪去之后，坐下来，和今天的自己说说话。

我开始学着观察那些曾被忽略的瞬间：清晨第一缕穿过窗帘的光，雨打在窗沿时细碎的声音，深夜码字时键盘清脆的敲击声。我希望用文字记录感受，让未来的我，能顺着这些碎片，拼凑起此刻真实活着的痕迹。

这个过程并不容易。习惯了奔跑的人，突然停下来，会感到眩晕。我常常对着空白的编辑器发呆，不知道该如何描摹这一段时间的经历与感受。但我相信，只要开始记录，就能慢慢填补记忆中那些苍白的缺口。

而我为何选择写博客呢？

因为，我相信写博客本身也是一种仪式感。选择一个主题，调整配色，搭建框架，写下第一行字，这整个过程像是在打造一个属于自己的世界。站点的每个细节都藏着我的偏执与审美，这本身也是一种表达。

给时间以生命，而不是给生命以时间。愿我不再追求塞满每一分钟，而是努力让每一分钟都值得被记住，把一切献给当下的每一个呼吸中。

# 未来

:::quote
往前看，别回头。
:::

关于未来，我并没有太多的规划。我想保持开放，像一张未曝光的底片，等待光线来作画。

我希望这片空间能陪我走下去。十年后，二十年后，当我再翻开这些文字时，能够会心一笑——原来当时的我是这样的，原来那些日子并没有白过。

也许未来的某一天，我会站在更高的地方回望此刻。那时我会感谢现在这个决定，感谢那个在深夜里敲下第一个字符的自己。

这世界很大，大到一个人的声音会被轻易吞没；这世界也很小，小到一间小屋就能装下一整个人生。

所以，只管写下去吧。把今天献给现在，把回忆留给未来。

往前看，别回头。

**新的旅程，开始了。**
